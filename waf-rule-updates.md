# CIS WAF rule change log 
{: #waf-rule-changelog}

Stay informed with daily updates to CIS WAF managed rules, including newly identified threats, detection improvements, ruleset changes, and enhanced security protections.
{: shortdesc}

This log records all newly released, updated, disabled, and emergency rule changes are they are published.

This topic includes:
* Newly released and updated managed WAF rules
* Rules deployed in logging-only (beta) mode
* Changes to default rule actions
* Disabled remediation rules
* Emergency rule releases outside the standard release cycle
* General WAF protection updates
